Fusion Gourmet - Asian Food Festival 
This is the official website for Fusion Gourmet, a gastronomic festival dedicated to Korean and Japanese cuisine. The project is a modern, elegant, and fully responsive landing page designed for lovers of Eastern culture.

Technologies Used
HTML5: Semantic structure for better accessibility and SEO.

CSS3: Custom styling using CSS Variables, Flexbox, and CSS Grid for a harmonic layout.

Google Fonts: Integration of Playfair Display and Lato typography for a premium aesthetic.

JavaScript: Implementation of basic interactions and navigation handling.

Design Features
Color Palette: Based on earth tones (Olive and Gold) over a deep black background to evoke sophistication and nature.

Responsive Design: Fully adapted for optimal viewing on mobile devices, tablets, and desktop computers.

Included Sections:

Hero Section: High-impact introduction with a clear Call to Action (CTA).

Schedule: Detailed table featuring dates, events, and international guest chefs.

The Festival: A space dedicated to the event's philosophy and its sponsors.

Gallery: A visual showcase of the main dishes on the menu.

Fusion Gourmet - Asian Food Festival - ESPAÑOL
Este es el sitio web oficial de Fusion Gourmet, un festival gastronómico dedicado a la cocina coreana y japonesa. El proyecto es una landing page moderna, elegante y totalmente responsiva diseñada para amantes de la cultura oriental.

Tecnologías Utilizadas
HTML5: Estructura semántica para una mejor accesibilidad.

CSS3: Diseño personalizado utilizando Variables CSS, Flexbox y CSS Grid.

Google Fonts: Tipografías seleccionadas (Playfair Display y Lato) para un look premium.

JavaScript: Manejo de interacciones básicas y navegación.


Características del Diseño
Paleta de Colores: Basada en tonos tierra (Oliva y Dorado) sobre un fondo negro profundo para evocar sofisticación.

Diseño Responsivo: Adaptado para dispositivos móviles, tablets y computadoras de escritorio.

Secciones Incluidas:

Hero Section: Introducción con llamada a la acción.

Agenda: Tabla detallada con horarios y chefs invitados.

The Festival: Información sobre la filosofía y patrocinadores.

Galería: Muestra visual de los platos principales.
